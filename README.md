## 🚀 Quick Start

### 📋 Prerequisites

```bash
# Required Software
Python 3.8+
Tesseract OCR
Poppler (for PDF support)
```

### ⚡ Installation (3 minutes!)

1. **Clone the repository**
```bash
git clone https://github.com/Nithiyasriramasamy/blood-test-medical-analizes.git
cd blood-test-medical-analizes
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Install Tesseract OCR**
   - **Windows**: [Download here](https://github.com/UB-Mannheim/tesseract/wiki)
   - **macOS**: `brew install tesseract`
   - **Linux**: `sudo apt-get install tesseract-ocr`

4. **Run the application**
```bash
python app.py
```

5. **Open in browser**
```
http://localhost:5000
```

### 🎯 First Use

1. **Upload** your medical report (PDF/JPG/PNG)
2. **Click** "Analyze Report" 
3. **Explore** 5 tabs of visualizations
4. **Chat** with AI assistant (click 💬 button)
5. **Ask** questions like "What is my health score?"

## 🎨 Customization

### 🔧 **Modify Reference Ranges**
Edit `reference_data.csv`:
```csv
Test_Name,Min_Value,Max_Value,Unit,Description
Hemoglobin,12.0,16.0,g/dL,Oxygen-carrying protein
```

### 🎨 **Change Colors**
Update `static/css/style.css`:
```css
:root {
  --primary: #667eea;
  --success: #2ecc71;
  --warning: #f39c12;
  --danger: #e74c3c;
}
```

### 🤖 **Add Chatbot Responses**
Extend knowledge base in `app.py`:
```python
knowledge_base = {
    'new_test': {
        'what': 'Explanation of new test...',
        'low': 'What low values mean...',
        'high': 'What high values mean...',
        'improve': 'How to improve...'
    }
}
```
